## Instalación

1. Clona este repositorio: `git clone https://github.com/LuciaRguez99/HTML-CSS_2DAW.git`
2. Navega al directorio del proyecto: `cd tu-proyecto`
3. Abre el archivo `advertencia.html` en tu navegador.

## Uso

Este proyecto ha sido creado para juntar todas las actividades y exámenes hechos durante el curso para que en el examen final sean más accesibles.

## Estructura del Proyecto

- `advertencia.html`: Archivo principal HTML.
- `components/`: carpeta que contiene los componentes que voy a utilizar en mi proyecto.
- `css/`: carpeta de estilos CSS.
- `img/`: carpeta que contiene las imágenes.
- `scripts/`: carpeta que guarda todos los scripts usados en el proyecto.
